The following files are attached to this manuscript:
\begin{itemize}
    \item \textbf{REINVENT4\_INPUTS.zip}: RL and TL input files used in REINVENT4 and initial REINVENT prior used in this work.
    \item \textbf{reinvent\_molecules.zip} CSV files with the SMILES codes of the compounds that passed the first filter at each iteration of the workflow. Contains RDkit descriptors, docking scores and CSA values, among other quantities of interest.
    \item \textbf{receptors.zip} PDB files of the receptor conformations used for docking.
\end{itemize}

The python and bash scripts used in this work are available at \textit{https://github.com/carajillu/genai} for information purposes. The material in this repository is currently in the process of being assembled into an open-source, ready-to-install tool.
